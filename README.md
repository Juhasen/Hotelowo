Project for ZAI/ORO course
# Hotel Search - Application Project

## Functional Scope

### User Roles
#### 1. Administrator
- Manage hotels
- Add/edit/delete hotels
- Manage user accounts
- View reservation statistics

#### 2. Standard User
- Search for hotels
- Filter results
- Make reservations
- Manage own reservations

## Key Features
### Basic
- Login and registration system
- User authentication (2 roles)
- Secured passwords
- Basic hotel management
- Search engine

### Extended
- Pagination of results
- Hotel image upload
- Form validation

### Advanced
- Internationalization (PL/EN)
- Dynamic form generation
- Responsive, aesthetic interface
- Review and rating system

## System Modules
### 1. Search Module
- Filters: location, price, ratings
- Sorting results
- Hotel details preview

### 2. Reservation Module
- Availability check
- Room selection
- Reservation cart
- Reservation history

### 3. User Module
- User profile
- Favorite hotels
- Notifications
